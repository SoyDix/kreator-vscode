# Kreator

**Kreator** is an open-source fork of Visual Studio Code that integrates an AI chatbot powered by OpenRouter directly into the editor.

It enhances the development experience by allowing you to interact with advanced language models without leaving your coding environment. The chatbot can help you write code, debug issues, explain logic, and refactor existing files using context from your workspace.

Kreator is built on top of the VS Code open-source codebase (`Code - OSS`) and maintains the familiar interface while adding native AI capabilities for a faster and more efficient workflow.
